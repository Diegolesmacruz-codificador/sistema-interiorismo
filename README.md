# Sistema Interiorismo

## Descripción

Sistema web desarrollado para la empresa **DL Interiorismo**, orientado a la gestión integral de clientes, empleados, proyectos, cotizaciones, facturación, pagos, tareas y reportes.

El proyecto fue desarrollado utilizando Spring Boot siguiendo una arquitectura por capas (Controller, Service, Repository y Model), aplicando buenas prácticas de programación, pruebas unitarias y análisis de calidad de código.

---

## Tecnologías utilizadas

- Spring Boot 3

## Arquitectura

El sistema está organizado mediante una arquitectura por capas.

```
Controller
      ↓
Service
      ↓
Repository
      ↓
Base de Datos
```

## Calidad del software

Durante el desarrollo se realizaron pruebas unitarias y análisis estático de código.

Herramientas utilizadas:

- JUnit 5
- Mockito
- JaCoCo
- SonarQube

Resultados obtenidos:

- Cobertura de pruebas: **97.2%**
- Código analizado con SonarQube
- Corrección de incidencias de cobertura
- Corrección de incidencias de mantenibilidad

## Cómo ejecutar el proyecto

1. Clonar el repositorio.

```
git clone https://github.com/Diegolesmacruz-codificador/sistema-interiorismo.git
```

2. Abrir el proyecto en IntelliJ IDEA.

3. Configurar la base de datos MySQL.

4. Ejecutar:

```
SistemaInteriorismoApplication
```
